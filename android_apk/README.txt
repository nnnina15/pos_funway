===== 風味食堂 POS — Android APK 打包說明 =====

不需要寫程式，用免費線上工具 3 步驟完成：

== 方法一：WebIntoApp（最簡單）==
1. 把 pos_funway.html 上傳到免費靜態空間，例如：
   - GitHub Pages（免費）
   - https://netlify.com（拖拉資料夾即可，免費）

2. 取得網址後（例如 https://yourname.netlify.app/pos_funway.html）
   前往：https://webintoapp.com
   輸入網址 → 設定 App 名稱「風味食堂POS」→ 下載 APK

3. 用 USB 傳到 SUNMI 機器，安裝即可

== 方法二：Netlify 部署（推薦）==
1. 前往 https://app.netlify.com/drop
2. 把 pos_funway.html 拖進去
3. 複製產生的網址
4. 在 SUNMI 瀏覽器直接開啟，或用上面方法包成 APK

== 方法三：直接在 SUNMI 瀏覽器設為首頁 ==
1. 用 USB 把 pos_funway.html 傳到 SUNMI
2. 用內建瀏覽器開啟
3. 瀏覽器選單 → 「設定為首頁」
4. 之後開瀏覽器就直接進 POS

== SUNMI 安裝 APK 步驟 ==
1. 設定 → 安全性 → 允許未知來源
2. 用檔案管理員找到 .apk 檔案
3. 點擊安裝
